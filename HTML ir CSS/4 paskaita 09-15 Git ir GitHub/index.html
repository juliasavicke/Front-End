Naudojant bet kuria isskyrus Static, gauname prieiga prie 4 nauju stiliu: top, left, right, bottom, z-index.
Sie stiliai nurodo per koki atstuma elementas turi buti atitrauktas nuo savo pradines pozicijos. Z-index stilius nurodo elemento prioriteta sluoksniuose

Static - nieko ypatingo nedaro, net nesutiekia galimybes naudoti top left right arba bottom, z-index.

Relative - nieko ypatingo nedaro, bet suteikia galimybe naudoti top right left bottom z-index. Elementui suteikus pozicija, jis atsitraukia per nurodyta atstuma nuo savo pradines pozicijos.

Absolute - elementas iseina is puslapio tekmes.
Elementui suteikus top right left bottom stilius jis atsitraukia per nurodyta atstuma nuo jo tevinio elemento turincio kazkuria pozicija, o jei tokio nera tuomet nuo html.

Fixed - elementas iseina is puslapio tekmes ir fiksuojasi nurodytoje arba savo pradineje pozicijoje.
Elementui suteikus top right left bottom stilius jis atsitraukia per nurodyta atstuma nuo html.
Taip pat, jeigu nera nustatytas elemento dydis - elementas sumazeja iki minimalaus dydzio. Naudojamas MENU.

Terminalo komandos

  • cd - change directory - pakeisti direktorija - vaiksciojimas tarp aplanku.
  • ls - list files and directories - isvardinti failus ir aplankus.
  • CTRL C - atsaukia vykdoma veiksma
  • TAB - uzbaigia fraze arba pateikia pasiulymus
  • clear - nukelia visa terminale rodoma informacija i virsu

git config --global user.name "username" - vartotojo vardas
git config --global user.email "email" - el.pastas
git config --global core.editor "code"
git config --global core.autocrlf true
git config --global -e - atidaro ir parodo info per editoriu
git config --help

Git komandos

  • git init - inicializuoti - sukuria git repozitoria.
  • git status - statusas - parodo dabartini tavo gito repozitorios statusa
  • git add - stage'ina/prideda/paruosia failus commit'inimui
  • git add . - paruosia visus failus commitinimui.
  • git commit -m "commit'o zinute" - sukuria kodo versija is stage'intu failu
  • push - lokalius (jau sucommit'intus) pakitimus perduoda GitHub'ui. test
