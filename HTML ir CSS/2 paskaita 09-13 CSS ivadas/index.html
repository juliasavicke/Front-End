Internal CSS - Vidinis CSS

Vienas id turi buti priskirtas vienam elementui ir vienas elementas turi tureti tik viena id.

Klase gali buti priskirta keliems elementams.

External CSS - Isorinis CSS

Isorinis stilizavimas naudojamas kaip taisyklingas budas stilizavimui.

CSS rasymo budai

  • Inline - linijinis rasomas tago viduje naudojant style atributa
  • Internal - vidinis rasomas style tago viduje head dalyje
  • External - isorinis rasomas atskirame CSS faile, kuri reikia prijungti prie HTML naudojant link taga

CSS Selectoriai

  • element - parenkamas elementas pagal jo pavadinima (p{})
  • id - sukuriamas susigalvotas id pavadinimas ir priskiriamas vienam elementui (#idpavadinias{})
  • class - sukuriamas susigalvotas klases pavadinimas ir priskiriamas bet kokiam kiekiui elementu (.klasesVardas{})

CSS Stiliai

  • color - teksto spalva (color: red)
  • font-size - teksto dydis (font-size: 20px)
  • background-color - fono spalva (background-color: green)
  • font-family - teksto sriftas (font-family: monospace)
  • font-weight - teksto storis (font-weight: bold)
  • line-height - teksto eilutes aukstis (line-height: 20px)
  • border - teksto remelis (border: 2px solid red)
  • border-radius - teksto remelio kampu apvalinimas (border-radius: 10px)
  • text-align - teksto lygiavimas (text-align: center)
  • font-style - teksto stilius (font-style: italic)
