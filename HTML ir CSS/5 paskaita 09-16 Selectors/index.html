CSS Selectors

Visus CSS selectors galima rasti cia

  • element - tagas
  • #id - id
  • .class - klase
  • .class1.class2 - kai suteiktos abi klases
  • .klase3 klase4 - kai elemento, turincio klase4 klase, teviniui elementui sutekta klase3 klase
  • * - suteikia viskam
  • element1, element2 - isvardintus ir kableliu atskirtus elementus
  • element1 > element2 - element2, kurio tevinis yra element1
  • a:link - nelankytos nuorodos
  • a:visited - lankytos nuorodos
  • a:hover - uzvedus pele
  • :nth-child(skaicius) - skaiciumi nurodytas kazkelintas arba kas kazkelintas vaikinis elementas
